## 🧩 CSS Grid & Flexbox: A Sinapse do Design

A interconectividade entre Grid e Flexbox permite a criação de interfaces complexas e responsivas.

- **Grid Layout**: Ideal para estruturas bidimensionais (linhas e colunas).
    - *Referência*: [Grid Layout](../04_Grid_Layout_Estruturas_Bidimensionais.md)
- **Flexbox**: Perfeito para alinhamento unidimensional de componentes.
    - *Referência*: [Flexbox](../03_Flexbox_Alinhamento_e_Distribuicao.md)

## ⚡ Variáveis CSS (Custom Properties)

As variáveis permitem uma rede de estilos dinâmica, conectando-se ao [Sistema de Cores](../../javascript/references/COLOR_SYSTEM.md) em JavaScript.

```css
:root {
  --primary-glow: #007bff;
  --aura-effect: 0 0 20px rgba(0, 123, 255, 0.5);
}
```
