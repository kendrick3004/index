# 🏗️ Estrutura Semântica e SEO em HTML

Este arquivo detalha a importância da semântica para a acessibilidade e o SEO, conectando-se diretamente com a [Sintaxe HTML](../01_Introducao_e_Sintaxe.md) e as [Meta Tags](../02_Meta_Tags_e_SEO_Basico.md).

## 🧠 A Sinapse da Estrutura

A interconectividade entre tags semânticas e o entendimento dos motores de busca é fundamental para o sucesso de um site.

- **Tags de Seção**: `<header>`, `<main>`, `<footer>`, `<section>`, `<article>`.
    - *Referência*: [Tags Semânticas](../03_Tags_Semanticas_e_Estrutura_de_Pagina.md)
- **Acessibilidade (ARIA)**: Conecta a estrutura visual com tecnologias assistivas.
    - *Referência*: [Acessibilidade Web](../06_Acessibilidade_Web_e_Padroes_W3C.md)

## ⚡ Otimização de Metadados

Metadados bem estruturados conectam o site com redes sociais e motores de busca, utilizando [Open Graph](../02_Meta_Tags_e_SEO_Basico.md).

```html
<meta property="og:title" content="Suite - Dashboard Pessoal">
<meta property="og:description" content="Dashboard com clima, liturgia e links úteis.">
```
